递归函数

递归函数优点逻辑简单清晰，缺点是调用过深会导致栈溢出，python标准解释器没有对针对尾递归做优化，任何递归函数都存在栈溢出的问题。

针对尾递归优化的语言可以通过尾递归防止栈溢出，尾递归事实上和循环是等价的，没有循环语句的编程语言只能通过尾递归实现循环。

来看一个阶乘的递归函数

理论上所有递归函数都可以写成循环的方式，但是，循环的逻辑不如递归清晰。使用递归函数需要注意的是栈溢出。在计算机中函数调用是通过栈（stack）这种数据结构实现的，每当进入一个函数调用，栈就会加一层栈帧，每当函数返回就会减一层栈帧，由于栈的大小不是无限的。解决递归调用的栈溢出的方法是通过尾递归优化。

尾递归是指，在函数返回的时候调用自身本身，并且，return语句不能包含表达式，这样，编译器或者解释器就可以把递归做优化，使递归无论调用多少次都只占用一个栈帧。上面的return n * fact(n-1)。

引入了表达式，所以就不是尾递归了，要改写成尾递归方式。

遗憾的是，大多数编程语言都没有针对尾递归优化，python也是，所以即使把上面的fact(n)函数改成尾递归方式也会导致栈溢出。
